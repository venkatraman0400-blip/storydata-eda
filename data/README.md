# Data

The raw data files are not tracked by git (they are too large and may have license restrictions).

## How to get the dataset

1. Go to [Kaggle Dataset Link Here](https://www.kaggle.com/)
2. Download the CSV
3. Place it in `data/raw/your_dataset.csv`
4. Run the notebooks in order starting from `01_act1_meet_the_data.ipynb`

## Folder structure

```
data/
├── raw/          ← place your downloaded Kaggle CSV here
└── processed/    ← auto-generated by notebook 01 after cleaning
```

The processed file is created automatically when you run Act 1.
